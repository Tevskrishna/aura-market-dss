# AURA-Market

**AI-Powered Predictive & Prescriptive Decision Support System for Urban Real Estate Analytics**

Micro-market focus: Bagaluru / Aerospace Highway (KIADB)

## Share with everyone (anytime)

1. Open [Streamlit Cloud Deploy](https://share.streamlit.io/deploy?repository=Tevskrishna/aura-market-dss&branch=master&mainModule=app.py)
2. Sign in with GitHub (`Tevskrishna`)
3. Confirm: repo **aura-market-dss**, branch **master**, main file **app.py** → **Deploy**
4. Wait 1–3 minutes → copy the public URL (looks like `https://….streamlit.app`)
5. Send this to anyone:

```
AURA-Market demo
URL: <paste your streamlit.app link>
Login: admin / admin123
(alt)  demo / demo123
```

Repo: https://github.com/Tevskrishna/aura-market-dss  

Your laptop does **not** need to stay on once Cloud is deployed.
